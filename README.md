# ArchP CPU

## Development

### Dependencies

- [veryl](https://github.com/veryl-lang/veryl)
- [just](https://github.com/casey/just)
- [xmake](https://github.com/xmake-io/xmake)
- [verilator](https://github.com/verilator/verilator)

### Work with Verilator

```bash
# Config
just config

# Build
just build

# Run
just run
```

### Work with Veryl tests

```bash
veryl test --wave
```
